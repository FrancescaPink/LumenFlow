# LumenFlow 🌊💡

LumenFlow is a platform designed to solve the "academic triage" problem: fitting study hours for multiple heavy exams into tight timeframes. When life happens and you miss a study day, LumenFlow dynamically recalculates your entire schedule, redistributing the remaining workload and eliminating the anxiety of manual rescheduling.

## 🛠 Tech Stack
* **Backend:** Python, Django
* **Frontend:** Islands Architecture with Vue 3 (CDN) and Tailwind CSS
* **Database:** PostgreSQL (Production) / SQLite (Development)
* **AI Engine:** LLM integration with a custom credit management and reward system

## ✨ Core Features
* **Adaptive Recalculation:** Optimization algorithms that automatically adjust your daily study goals based on missed days or changed deadlines.
* **"Lazy" Credit Management:** A zero-cost backend approach to handle daily AI API limits without background workers.
* **Integrated Monetization:** Built-in logic for Rewarded Ads to unlock AI features, balancing free access with server cost sustainability.
